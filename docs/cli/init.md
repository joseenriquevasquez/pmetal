# pmetal init

Generate a sample YAML configuration file.

Generate a sample `pmetal.yaml` configuration file with documented defaults.

## Usage

```bash
pmetal init [OPTIONS]
```

## Examples

```bash
# Generate default config
pmetal init

# Specify output path
pmetal init --output my-config.yaml
```
