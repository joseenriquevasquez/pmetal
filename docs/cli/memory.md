# pmetal memory

Show memory usage and available capacity.

Display current memory usage and available capacity on your Apple Silicon device.

## Usage

```bash
pmetal memory
```

## Output

Shows unified memory allocation including GPU, ANE, and system usage with available capacity for model loading and training.

- [pmetal info](/cli/info/) — Full device info
- [pmetal search](/cli/search/) — Memory fit estimation for models
